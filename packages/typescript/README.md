# eslint-config-typescript

Sharegate TypeScript ESLint config.

## Usage

If you are also using React, you might have to add the following ESLint rule:

```js
rules: {
    "react/jsx-filename-extension": [1, { "extensions": [".js", ".jsx", ".ts", ".tsx"] }]
}
```
